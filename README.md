Telegram бот для размещения объявлений

Telegram бот для размещения профессиональных объявлений с системой модерации. Пользователи могут отправлять свои профессиональные анкеты, которые проверяются модератором перед публикацией в канале.

---

## 📋 Возможности

### Для пользователей:
- ✅ Отправка профессиональных объявлений с личной информацией
- ✅ Просмотр информации о проекте
- ✅ Получение инструкции по использованию бота
- ✅ Связь с поддержкой
- ✅ Получение уведомлений о статусе поста (опубликован/отклонён)

### Для модератора:
- ✅ Просмотр входящих объявлений
- ✅ Принятие или отклонение постов с комментариями
- ✅ Ответы на запросы в поддержку
- ✅ Публикация одобренных постов в канале
- ✅ Отклонение постов с возможностью оставить отзыв

---

## ⚙️ Установка и настройка

### 1. Клонирование репозитория

```bash
git clone <repository-url>
cd <project-directory>
2. Установка зависимостей
bash
pip install -r requirements.txt
3. Настройка переменных окружения
Создайте файл .env в корневой директории проекта со следующими переменными:

env
BOT_TOKEN=your_telegram_bot_token
ADMIN=your_telegram_user_id
CHANNEL=your_channel_username
BOT_TOKEN: Получите у @BotFather в Telegram

ADMIN: Ваш Telegram ID (можно получить у @userinfobot)

CHANNEL: Имя пользователя канала (например, @vprofi_ingushetia)

4. Добавление бота в канал
Добавьте бота как администратора в канал, где будут публиковаться посты. Боту необходимо разрешение на отправку сообщений.

⚠️ Важно
Убедитесь, что бот настроен как администратор канала перед запуском. Бот должен иметь права на отправку сообщений в канал.

Проверка перед запуском:
□ Бот добавлен в канал как администратор
□ Бот имеет права на отправку сообщений
□ Файл .env создан и заполнен
□ Установлены все зависимости
□ Канал существует и доступен
